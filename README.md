# 🚀 Dockerized Flask Application

## 📌 Project Overview
This project demonstrates the containerization of a simple Flask web application using Docker. It showcases how an application can be packaged with its dependencies and deployed consistently across different environments.

---

## 🛠️ Tech Stack
- Python (Flask)
- Docker

---

## ⚙️ Features
- Lightweight Flask web application
- Dockerized for consistent deployment
- Runs inside a container with exposed port
- Easy to build and run locally
